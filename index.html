Draw your site on the map — generates Litchi Hub CSV files for 3DGS drone capture · Missions A (grid) · B (orbit) · C (perimeter)

1 — Capture Area

Draw your site: Use the polygon or rectangle tool on the map toolbar (top-left). Click to place each corner, double-click to finish. Use the edit/drag tools to adjust. Search for your location below.
    Overlays: Grid A Grid B Orbit Perimeter Centre D
    Import / export coordinates (Google Maps, paste lat,lng)
    Right-click any corner on Google Maps → What's here? → tap the lat,lng to copy. Paste below, one pair per line. Changes sync to the map automatically.
    Clockwise or counter-clockwise — both work. Editing the map updates this automatically.
